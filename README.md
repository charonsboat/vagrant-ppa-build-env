# vagrant-ppa-build-env

A build environment for [PPAs](https://launchpad.net/ubuntu/+ppas). Specifically, [Vagrant](https://www.vagrantup.com/).


## getting started

To get started, you will need Vagrant and [VirtualBox](https://www.virtualbox.org/) (or another VM manager) installed. Then, simply clone this project, launch it with Vagrant, and ssh into the new machine:

```bash
git clone https://github.com/drmyersii/vagrant-ppa-build-env.git
cd vagrant-ppa-build-env
vagrant up
vagrant ssh
```
